# FadeStep Starter Site

A simple static website for a backward-faded worksheet idea.

## Pages
- `index.html` — homepage
- `bank.html` — starter worksheet bank
- `generator.html` — browser-based backward fading generator
- `styles.css` — site styling
- `script.js` — generator logic

## Deploy on GitHub Pages
1. Create a new GitHub repository.
2. Upload all files in this folder.
3. In the repository, go to **Settings > Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select your main branch and the root folder.
6. Save.
7. GitHub will publish the site and give you a Pages URL.
